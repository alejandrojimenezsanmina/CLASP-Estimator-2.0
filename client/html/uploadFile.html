How to mass upload?
You can create a file (google or excel) with the required information shown in the header of the embeded google sheet.
Paste the required information into the sheet (be careful not to override the header).
⚠ Some columns require specific information and there are validation rules such as material, finish, etc...



⚠ Make sure to select finish type 1 and 2, these are required in order to generate estimations.
If there is no finish required, select "No finish required" in both columns



Click on estimate and the result of your estimations will be done in a couple of seconds...
